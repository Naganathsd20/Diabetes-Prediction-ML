# 🩺 Diabetes Prediction Using Machine Learning

## 📌 Overview

This project is a Machine Learning-based Diabetes Prediction System developed using Python and the Support Vector Machine (SVM) algorithm. It predicts whether a patient is diabetic or non-diabetic based on medical attributes.

This project was completed as part of my AI/ML internship at InternPe.

## 🚀 Features

- Data Loading and Exploration
- Data Visualization
- Missing Value Analysis
- Feature Standardization
- Train-Test Split
- SVM Model Training
- Model Evaluation
- Diabetes Prediction using Custom Input

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## 📂 Project Structure

- `Diabetes_Prediction.ipynb` – Complete Machine Learning notebook
- `diabetes.csv` – Dataset used for training and testing
- `README.md` – Project documentation

## 📈 Workflow

1. Import Libraries
2. Load Dataset
3. Explore Dataset
4. Visualize Data
5. Data Preprocessing
6. Standardize Features
7. Train-Test Split
8. Train SVM Model
9. Evaluate Model Accuracy
10. Predict Diabetes Using Custom Input

## 🎯 Output

The trained model predicts whether a patient is:

- Non-Diabetic
- Diabetic

based on the provided medical information.
